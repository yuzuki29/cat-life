# レスポンシブ対応
✅ **完全レスポンシブ対応を実装しました**

## 対応ブレークポイント
- **超小型スクリーン (320px～375px)** - スマートフォン
  - フォントサイズを最小化
  - パディング・マージンの最適化
  - キャラクター要素のサイズ調整

- **小型スクリーン (376px～576px)** - 大型スマートフォン
  - ヘッダーの縮小
  - フレックスレイアウトの柔軟化
  - ボタン・フォーム要素の最適化

- **タブレット (577px～768px)** - iPadなど
  - 2列レイアウトの適用
  - パディング調整
  - 画像の高さ最適化

- **大型タブレット/小型デスクトップ (769px～992px)** - iPad Pro など
  - 3列レイアウトの準備

- **デスクトップ (993px以上)** - PC
  - フルレスポンシブデザイン

## 対応コンポーネント
- ✅ ナビゲーションバー (モバイルメニュー対応)
- ✅ ヒーローセクション (テキスト・画像のリサイズ)
- ✅ カード要素 (グリッドレイアウト)
- ✅ フォーム要素 (入力欄・ボタンの最適化)
- ✅ テーブル (スクロール対応)
- ✅ アコーディオン (モバイル最適化)
- ✅ フッター (カラム調整)
- ✅ キャラクター表示 (サイズ調整)

# カーソルを追いかける猫
・カーソルを上下左右に動かすとねこがカーソルを追いかけてくる  
・カーソルが止まると待機モーションに入る  
・クリック又はctl+クリックをすると猫が攻撃をする  
・画面内を移動する敵を倒すと猫が進化して攻撃パターンが変化する
