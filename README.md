# サイトの概要
猫との暮らしをサポートするためのサイトで、猫の飼い方、健康管理、しつけの仕方を詳しく解説

# デザインの変更点
サイドバーにあった質問投稿ボタンと人気の質問を横いっぱいに広げて投稿された質問の下に移動した
人気の質問は表示する数を増やした

#　追加ページ
各猫の詳細ページ、ログインページ、新規登録ページ、Q&Aの質問投稿ページ、お問い合わせページ

# カーソルを追いかける猫
・カーソルを上下左右に動かすとねこがカーソルを追いかけてくる  
・カーソルが止まると待機モーションに入る
・クリック又はctl+クリックをすると猫が攻撃をする
・画面内を移動する敵を倒すと猫が進化して攻撃パターンが変化する
