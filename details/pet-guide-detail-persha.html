中級者向け

ペルシャ

「猫の王様」。豪華な長毛と優雅な雰囲気が特徴の、非常に穏やかな愛玩猫

基本情報

サイズ
中〜大型
体重
3-5.5kg
寿命
14-17年
原産地
イラン（ペルシャ）

ペルシャ猫は、その豪華な長毛と短く平らな鼻を持つ顔立ちで「猫の王様」として世界中で愛されています。 非常に穏やかで、ほとんど鳴くことがなく、おとなしい性格です。手入れには手間がかかりますが、その優雅な姿は多くの人を魅了します。

性格
  • 非常に穏やか : ほとんど鳴かず、物静かで落ち着いている
  • 愛玩猫の鏡 : 飼い主のそばでくつろぐのを好む
  • マイペース : 活発な遊びより、寝て過ごすことが多い
  • 協調性 : 他のペットや子供とも比較的仲良くできる
身体的特徴
  • 短く太い手足、ずんぐりとしたコビータイプの体型
  • 豪華で絹のような超長毛（ダブルコート）が全身を覆う
  • 鼻が低く平たいピークフェイスが特徴（一部の系統を除く）
  • 涙やけが起こりやすい短頭種

日常のケア
  • 運動 : 激しい運動は不要。低いキャットタワーや寝床を用意
  • ブラッシング : 毎日欠かさず行う。怠ると毛玉になり皮膚炎の原因に
  • 顔の手入れ : 涙やけを防ぐため、毎日目の周りを拭く
  • 室温管理 : 長毛と短頭種であるため、夏の冷房による温度管理が必須
食事
  • 肥満を防ぐための適切なカロリー管理が重要
  • 平たい顔でも食べやすいように、キブル（粒）の形や大きさに配慮されたフードが推奨
  • 長毛種のため、毛玉ケア用のフードを併用する

かかりやすい病気
  • 多発性嚢胞腎 (PKD) : 遺伝性疾患。腎臓に嚢胞ができる
  • 肥大性心筋症 (HCM) : 遺伝的要因が高い心臓病
  • 毛球症 : 長毛のため、毛づくろいによる毛の飲み込みが多い
  • 流涙症・皮膚炎 : 短頭種特有の涙やけや顔周りの皮膚トラブル
予防と対策
  • 毎日のブラッシングと毛玉ケアフードの徹底
  • 遺伝性疾患（PKD/HCM）の定期的な検査（特にエコー検査）
  • 目の周りの清潔を保つ（涙やけ対策）
  • 適切な体重管理で心臓への負担を減らす

しつけのポイント
  • 落ち着いた性格で、しつけは容易な部類
  • トイレや爪とぎは基本的にすぐに覚える
  • 長毛ケアをスムーズにするため、子猫期からのスキンシップとブラッシングの練習が最も重要
基本的なしつけ
  • 穏やかな環境で、過度な刺激を与えないようにする
  • ブラッシングやシャンプーを嫌がらないように、褒めながら行う
  • 飼い主の根気強い手入れと愛情が信頼関係を築く鍵となる
