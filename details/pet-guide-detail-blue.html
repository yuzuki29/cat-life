ロシアンブルー

「ボイスレスキャット」と呼ばれる静かで賢い猫。シルバーブルーの被毛と緑の瞳が特徴

基本情報

サイズ
中型
体重
3-5kg
寿命
13-16年
原産地
ロシア

ロシアンブルーは、ロシアのアルハンゲリスク島が原産とされる、美しい銀青色の被毛を持つ猫種です。 非常に静かで鳴き声が小さいため「ボイスレスキャット」とも呼ばれます。賢く穏やかな性格で、控えめながらも飼い主に対して深い愛情を示します。

性格
  • 賢く穏やか : 環境やルールをすぐに理解する、落ち着いた性格
  • ボイスレスキャット : ほとんど鳴かない、または非常に小さい鳴き声
  • 人見知り : 飼い主には愛情深いが、知らない人には警戒心が強い
  • プライベート好き : 一人で過ごす時間も大切にする
身体的特徴
  • しなやかでフォーリンタイプの引き締まった体型
  • 光沢のある短毛（ダブルコート）で、被毛の先端が銀色に輝くティッピングが特徴
  • 瞳は成長と共にエメラルドグリーンに変化する
  • すらりとした体型と、特徴的な「ロシアン・スマイル」と呼ばれる口角

日常のケア
  • 運動 : 活発で遊び好き。おもちゃやキャットタワーで適度な運動を確保
  • ブラッシング : 週に1〜2回のコーミングで十分
  • 環境 : 静かな環境を好み、隠れる場所（ハウス）を用意してあげる
  • 留守番 : 一人でのお留守番が得意な猫種である
食事
  • 健康維持のための適切なカロリー管理が重要
  • 短毛種用の一般的なキャットフードで問題ない
  • 美しい被毛を保つために、オメガ脂肪酸を含むフードも推奨

かかりやすい病気
  • 尿路疾患（膀胱炎・結石） : ストレスや飲水量が少ないとリスクが高まる
  • 肥満 : 運動量が低下すると体重が増えやすいため注意が必要
  • 皮膚疾患 : 比較的稀だが、アレルギー等で皮膚炎になることがある
予防と対策
  • 飲水量を増やす工夫（新鮮な水、水飲み場を増やす）
  • 遊びを通して適度な運動をさせ、体重管理を徹底する
  • 遺伝性疾患のリスクは低めだが、定期的な健康診断を行う

しつけのポイント
  • 非常に賢いため、基本的なしつけは容易
  • トイレや爪とぎの場所をすぐに覚える
  • 環境の変化や来客に慣れさせるための子猫期の社会化が重要
基本的なしつけ
  • 静かな場所にトイレや食事のスペースを確保する
  • 遊びを十分に行い、愛着関係を深める
  • 単独飼育、静かな家庭に向いている
