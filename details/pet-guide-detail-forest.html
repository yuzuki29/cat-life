ノルウェージャンフォレストキャット

北欧の厳しい寒さで育った、フサフサの被毛と穏やかな性格の大型猫

基本情報

サイズ
大型
体重
4-8kg
寿命
14-16年
原産地
ノルウェー

ノルウェージャンフォレストキャットは、名前の通りノルウェーの森で自然発生した猫種です。 特徴的なダブルコートの長毛を持ち、非常に丈夫で病気になりにくいとされています。 穏やかで賢い性格から、"優しい巨人"とも呼ばれます。

性格
  • 穏やか : 滅多に鳴かず、落ち着いている
  • 賢い : 環境への順応性が高く、しつけやすい
  • 友好的 : 子供や他のペットに対しても寛容
  • 遊び好き : 特に高い場所への昇り降りが好き
身体的特徴
  • 大型で筋肉質な体つき
  • 分厚いダブルコートの長毛とフサフサの尻尾
  • 耳の先端にあるタフト（飾り毛）が特徴
  • 成猫になるまでに3～5年かかる

日常のケア
  • 運動 : 活発なので、縦の運動ができる高いキャットタワーが必要
  • ブラッシング : 毎日行う。特に換毛期は入念に
  • 温度管理 : 長毛のため、日本の夏は冷房による室温管理が必須
  • シャンプー : 月1回程度（長毛のため手間がかかる）
食事
  • 大型猫用・長毛種用のフードを選ぶ
  • 長毛のため、毛玉ケア用のフードも併用推奨
  • 肥満防止のため、給餌量を厳密に管理する

かかりやすい病気
  • 肥大性心筋症 (HCM) : 大型猫に多い心臓病
  • 多発性嚢胞腎 (PKD) : 腎臓に嚢胞ができる遺伝性疾患
  • 毛球症 : 長毛種のため、毛づくろいで毛を飲み込みやすい
予防と対策
  • 毛球症予防のため、ブラッシングと毛玉ケアフードを徹底
  • 定期的な心臓のエコー検査（HCMの早期発見）
  • 適切な体重管理と涼しい環境の維持
  • 飲水量を増やす工夫をする

しつけのポイント
  • 賢く協調性があるので、基本的なしつけは容易
  • 叱るよりも褒めて伸ばすポジティブトレーニングが有効
  • 家族全員が愛情をもって接することが大切
基本的なしつけ
  • 高い場所に登るための場所を明確にしてあげる
  • ブラッシングや爪切りに慣れさせるための子猫期からのトレーニング
  • 遊びの時間を確保し、運動不足を防ぐ
